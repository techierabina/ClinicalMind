# ClinicalMind
ICU Mortality Prediction using MIMIC-IV

## Overview
A multi-modal ICU mortality prediction system using MIMIC-IV clinical data, PyTorch, BioBERT, ChromaDB, and FastAPI.

## Data Access
Data is from MIMIC-IV via PhysioNet (credentialed access required).
Visit: https://physionet.org/content/mimiciv/3.1/
Do NOT commit any data files.

## Project Structure
- `data/` — local data only, not committed to GitHub
- `notebooks/` — exploration and analysis notebooks
- `src/` — model code
- `models/` — saved model checkpoints

## Tech Stack
- PyTorch
- BioBERT
- ChromaDB
- FastAPI
